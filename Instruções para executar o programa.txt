Após baixar o arquivo do repositório, é necessário executar o comando "composer install" no terminal.
Também é necessário configurar corretamente o arquivo /config/app_local.php para conseguir fazer a conexão com o banco de dados. No meu caso, foi necessário alterar "port" para 3307.
O usuário criado para o banco de dados tem como "username" = "geekstop" e "password" = "geekstop".
